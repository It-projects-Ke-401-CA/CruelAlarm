Принцип

В начале работы нужно создать будильник и сфотографировать несколько предметов в своей квартире. При срабатывании будильник отображает одну из фотографий, её нужно повторить для отключения звонка.

Создание будильника

Новый будильник можно создать кнопкой на главном экране приложения. По умолчанию будильник срабатывает только один раз, для регулярной работы нужно выбрать дни в раскрывающемся окне будильника.

Добавление фотографий

Чтобы сделать новый снимок, нужно выбрать Настройки>Менеджер фотографий и использовать кнопку в нижнем правом углу. Добавление снимков из памяти устройства не поддерживается.

Сравнение фотографий выполняется по участкам контрастного цвета. Если на фотографии таких участков слишком мало, она будет отвергнута.

Пробуждение

Будильник при запуске выбирает и показывает случайную фотографию из отмеченных галочками в Менеджере фотографий. Из открытого окна можно перейти в режим камеры, чтобы повторить снимок. Сравнение сделанного снимка с исходным занимает несколько секунд, затем будет показан результат.

Когда сделанный снимок принят, можно отключить будильник или повторить всё заново через время, указанное в настройках (5 минут по умолчанию).

Математические примеры

Если повторить фотографию не получается, можно перейти к решению математических примеров c помощью кнопки-креста на экране с результатом.

Кнопка Назад позволяет вернуться от примеров к попыткам сделать фотографию.

Если фотографии не принимаются

Программа корректирует сдвиги по горизонтали и вертикали, поэтому точное повторение исходного снимка не обязательно. Однако требуется сохранить ракурс и цветовую температуру исходной фотографии.

Для хорошего распознавания фотографируйте предметы ярких цветов с такого расстояния, чтобы они целиком входили в кадр (но не терялись на снимке).
